To run the code, you will need the following packages in addition to Python 3.7 and jupyter:
hiive-mdptoolbox
gym

1. Open a jupyter notebook instance, after downloading and unzipping the entire folder 'HW4'.
2. The hiive-mdptoolbox needed to run the forest problem is also part of the folder. Install using the setup file provided inside the folder hiivemdtoolbox.
3. gym-mod contains the modified version of OpenAI gym for the modified runs of frozen lake. Install using the setup file inside the folder gym-mod.
4. Open Assignment #4.ipynb inside the hiivemdptoolbox folder.
5. Once jupyter notebook is open, you should be able to run each cell directly.
